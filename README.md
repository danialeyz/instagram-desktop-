# Instagram Desktop Home Page (Static Version)

This project is a static version of the Instagram desktop home page, created as a practical project for a front-end development bootcamp. It showcases the design and layout of the Instagram home page, replicating its appearance and functionality using HTML and CSS.

## Features

- **Replicated Design:** The project accurately replicates the design and layout of the Instagram desktop home page.
- **HTML and CSS:** The project is built using HTML and CSS, making it easy to understand and modify.
- **Static Version:** This is a static version of the Instagram home page, meaning it does not include dynamic functionality or server-side interactions.
- **Responsive Design:** The page is designed to be responsive, ensuring optimal viewing experience across different devices and screen sizes.

## Technologies Used

- HTML
- CSS

## Usage

To view the Instagram desktop home page static version, simply open the `index.html` file in a web browser.
